# Files

Once a file share is mounted as a network drive on a server and registered within the platform, its content gets automatically
[indexed](../../access/files-indexer.md) and can be browsed.

## Connection parameters

```json
{
  "parameters": {
    "dir": "",
    "indexFiles": true
  },
  "credentials": {
    "parameters": {
      "login": "",
      "password": ""
    }
  }
}
```
