# Hive2

Provides access to [Apache Hive](https://hive.apache.org/) NoSQL database using
SQL queries via HiveSever2 JDBC driver.

## Connection parameters

```json
{
    "server": "",
    "port": "",
    "db": "",
    "connString": ""
}
```
