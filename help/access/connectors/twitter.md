---
title: "Twitter"
---

Provides ability to run search via [Twitter](https://twitter.com/) posts by search key and geocode parameters and return
results in tabular form.

## Connection parameters

```json
{
  "apiKey": "",
  "apiSecret": "",
  "accessToken": "",
  "accessSecret": ""
}
```
