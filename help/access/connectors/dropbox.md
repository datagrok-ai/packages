# Dropbox

Provides access to [Dropbox](https://www.dropbox.com) storage as [file share](files.md).

## Connection parameters

```json
{
  "parameters": {
    "dir": ""
  },
  "credentials": {
    "parameters": {
      "password": ""
    }
  }
}
```
