# BigQuery

Provides access to [Google BigQuery](https://cloud.google.com/bigquery/) database using SQL queries via JDBC driver.

## Connection parameters

```json
{
  "parameters": {
    "connString": "",
    "projectId": ""
  },
  "credentials" : {
    "parameters": {
      "login": "",
      "password": ""
    }
  }
}
```
