---
title: "PI"
---

Provides access to the
[PI](https://techsupport.osisoft.com/products/) database using SQL queries via JDBC driver.

## Connection parameters

````json
{
   "parameters": {
    "accessServer": "",
    "port": port,
    "db": "db-name",
    "cacheSchema": false,
    "cacheResults": true,
    "ssl": false,
    "connString": ""
  }
}
````
