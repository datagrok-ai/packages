---
title: "Azure Blob"
---

Provides access to the
[Azure](https://azure.microsoft.com/en-us) as
[file share](../../files/files.mdx).

## Connection parameters

````json
{
  "parameters": {
    "account": "",
    "container": "",
    "use SAS": "",
  },
  "credentials": {
    "parameters": {
      "shared token": ""
    }
  }
}
````
